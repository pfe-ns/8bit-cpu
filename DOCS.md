# ALU consists of 6 signals
*alu_input_1 - first ALU input ( 4 bit)
*alu_input_2 - second ALU input ( 4 bit)
*alu_output - output signal of ALU ( 4 bit)
*busy - one bit signal that tells us if ALU is busy or not
*clk - clock input
